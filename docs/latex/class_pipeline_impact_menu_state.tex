This method prompts the user to input a single character representing their choice in the Pipeline Impact Menu. It uses a switch statement to determine the action corresponding to the user\textquotesingle{}s choice. If the input is invalid, the method notifies the user and prompts them to try again. The \textquotesingle{}q\textquotesingle{} option exits the program.

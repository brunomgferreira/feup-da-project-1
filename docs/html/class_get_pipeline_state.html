Constructs an instance of GetPipelineState with specified back state and callback function.

This constructor initializes an instance of the GetPipelineState class with the given back state and a callback function for transitioning to the next state. The back state represents the state to which the application should return when the user chooses to go back from the current state. The callback function specifies the action to be performed in the next state, usually involving capturing the pipeline code from user input and proceeding to the next state with the captured information.

Handles user input for obtaining a pipeline code.

This method reads a line of input from the console, representing a pipeline code. The method validates the existence of the entered pipeline code in the application's data. If the pipeline code is valid, the callback function is invoked with the captured pipeline code, and the application transitions to the next state accordingly. If the pipeline code is invalid, the user is prompted with an error message, and the state transitions to a "Try Again" state, allowing the user to make another attempt.
